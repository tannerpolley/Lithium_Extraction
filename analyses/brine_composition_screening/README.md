# Brine Composition Screening

This analysis compares common lithium-bearing brine classes and screens produced-water candidates for the Rezaee 90 wt pct DES plus 10 wt pct TOPO Li/Na extraction basis.

The generated report is audience-facing: it avoids local path language and states source status, uncertainty, and screening limitations directly.

## Run

```powershell
uv run python analyses\brine_composition_screening\scripts\build_brine_screening_study.py
```
