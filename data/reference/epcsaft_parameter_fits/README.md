# ePC-SAFT Parameter Fit Diagnostics

This folder stores diagnostic parameter-regression outputs that are useful evidence for future ePC-SAFT parameter work but are not complete package-ready datasets.

- `rezaee_2026/` contains a DES pseudo-component fit used as a regression smoke test and package-plumbing diagnostic.

Move a fit into `data/reference/epcsaft_parameters/<DatasetName>/` only after it has a complete documented runtime payload with pure parameters, user options, and any required interaction tables.
