# Extraction Model Reference Payloads

This folder stores reusable source-regressed extraction-model payloads that are not complete ePC-SAFT pure-component or mixture parameter datasets.

- `gando_2025/` contains the HBTA/TOPO Li/Na staged-transfer model fitted from source literature anchors. It is suitable for the current case-study bridge and presentation tables.

Do not treat these files as package-ready ePC-SAFT parameters for HBTA, TOPO, diluent, or Li-organic complexes. Those missing components still need direct parameter curation or regression before a full predictive reactive ePC-SAFT LLE model can be claimed.
