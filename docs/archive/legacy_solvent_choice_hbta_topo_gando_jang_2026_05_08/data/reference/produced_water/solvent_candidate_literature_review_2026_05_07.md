# Non-Ionic Lithium Solvent-Extraction Candidate Scorecard

Generated by `scripts/case_study/solvent_candidate_scorecard.py`.

## Scope

Ionic-liquid extraction systems are excluded from the active candidate ranking because the current case-study direction treats them as too costly and uncertain for the pitch. Ionic-liquid/ePC-SAFT papers remain useful only as method references for parameter fitting and electrolyte-LLE diagnostics.

The ranking separates three questions that were previously mixed together:

1. Is the solvent chemistry source-backed for produced-water lithium recovery?
2. Can the chemistry be run today in this repository?
3. Would a completed ePC-SAFT implementation give PrOMMiS/IDAES better transfer variables than a single source-regressed Li/Na model?

The priority order is presentation-driven rather than a pure descending numeric score: Rezaee 2026 is now the flagship Li/Na bridge because it has the strongest package-facing parameter, reaction, extraction-response, and equilibrium-composition evidence. HBTA/TOPO remains the best conventional field-water comparison and future parameterization lane.

## Candidate Notes

### 1. TBAC + decanoic-acid DES + TOPO

- **Sources:** Rezaee 2026. DOI/URL: `10.1016/j.fluid.2026.114737`.
- **Evidence summary:** Current flagship Li/Na bridge because the paper reports DES/TOPO/RLi/RNa PC-SAFT-style values, organic binary interactions, reaction constants, extraction-response targets, and SI equilibrium-composition rows. The repo now derives distribution coefficients and Smackover Na/TDS sensitivity rows for PrOMMiS/IDAES from this source-backed basis.
- **Best available run:** `uv run python analyses\rezaee_2026_pcsaft_epcsaft\scripts\rezaee_li_na_distribution_bridge.py`.
- **Artifacts:** `rezaee_li_na_distribution_coefficients.csv; rezaee_smackover_tds_na_sensitivity.csv; rezaee_prommis_idaes_transfer_handoff.csv; rezaee_idaes_costing_input.csv`.
- **Next data needed:** accepted direct ePC-SAFT LLE phase splits across the surrogate domain, stronger TDS-dependent electrolyte objective, and approved DES/TOPO solvent-loss and cost assumptions

### 2. HBTA + TOPO + sulfonated kerosene

- **Sources:** Gando-Ferreira/Shan 2025 field-water paper; Zhang 2017 HBTA/TOPO stoichiometry; Zhang 2018 process support. DOI/URL: `10.3390/w17152258; 10.1016/j.seppur.2017.07.028; 10.1016/j.hydromet.2017.10.029`.
- **Evidence summary:** Only candidate with actual oil-and-gas field-water extraction evidence plus conventional non-ionic ligand chemistry. The repo has a source-regressed Li/Na staged model and IDAES costing handoff, but this is no longer the active flagship bridge for the Phase 0-9 TEA story. Zhang 2017 supports 2 HBTA : 1 TOPO : 1 Li stoichiometry; Zhang 2018 supports multistage HBTA/TOPO/kerosene operation.
- **Best available run:** `uv run python scripts\case_study\hbta_topo_reactive_stage_solve.py`.
- **Artifacts:** `hbta_topo_reactive_model_report.md; hbta_topo_reactive_stage_results.csv; smackover_prommis_transfer_handoff.csv`.
- **Next data needed:** HBTA, TOPO, sulfonated-kerosene/diluent, Li-BTA-TOPO parameters; binary interactions; reaction constants or digitized extraction curves for fitting

### 3. D2EHDTPA + BuPhen + octanol modifiers + n-dodecane

- **Sources:** Raiguel 2024. DOI/URL: `10.1039/d4gc04760e`.
- **Evidence summary:** Strong conventional solvent-extraction selectivity over Na, K, Mg, and Ca in synthetic geothermal brine. It is a serious backup chemistry because it directly targets multication brines, but the ligand/synergist set is less simple than HBTA/TOPO and is not an oil-and-gas produced-water source.
- **Best available run:** `parameter-availability check only`.
- **Artifacts:** `solvent_candidate_run_matrix_2026_05_07.csv`.
- **Next data needed:** pure-component parameters or density/vapor-pressure/liquid-density data for D2EHDTPA, BuPhen, octanol modifiers, and n-dodecane; extraction-equilibrium curves for apparent reaction constants

### 4. TBP + FeCl3/HCl

- **Sources:** Kia 2024. DOI/URL: `10.1007/s11356-024-34617-8`.
- **Evidence summary:** Useful non-ionic comparison for high-Mg brines with reported Li/Mg separation and explicit Fe-chloride extraction chemistry. It is less attractive for the pitch because it brings acid, iron handling, third-phase risk, and corrosion/process burdens.
- **Best available run:** `parameter-availability check only`.
- **Artifacts:** `solvent_candidate_run_matrix_2026_05_07.csv`.
- **Next data needed:** TBP and Fe/Li chloride complex parameters, extraction-equilibrium constants, aqueous Fe/chloride speciation basis, and acid/corrosion cost assumptions

### 5. D2EHPA + TBP

- **Sources:** Jang 2017; Lee 2020. DOI/URL: `10.1016/j.apgeochem.2017.01.016; 10.1016/j.apgeochem.2020.104571`.
- **Evidence summary:** Real shale-gas produced-water context and useful for showing why a generic solvent-extraction placeholder is not enough. The current local script predicts weak Li/Na discrimination and is best used as a comparison/limitation slide.
- **Best available run:** `uv run python scripts\lle\jang_2017_stage2_li_na_tbp_d2ehpa.py (may be slow; existing output retained if timeout occurs)`.
- **Artifacts:** `jang_2017_stage2_li_na_summary.md; jang_2017_stage2_li_na_summary.csv`.
- **Next data needed:** true D2EHPA, TBP, diluent, acid/base reaction constants, and produced-water organic-contaminant impacts

## Recommended Presentation Position

Use Rezaee 2026 DES/TOPO as the flagship Li/Na case-study bridge for the current Phase 0-9 TEA story because it supplies source-backed Li/Na extraction responses, SI equilibrium-composition rows, organic PC-SAFT parameter payloads, binary interactions, and reaction constants. The generated bridge now converts those targets into distribution coefficients, Smackover Na/TDS sensitivity rows, and PrOMMiS/IDAES transfer variables.

Use HBTA/TOPO/sulfonated kerosene as the comparison and future parameterization lane. It remains important because it has field-water and conventional-ligand relevance, but it should not be the active flagship until the organic pure-component, complex, binary-interaction, and reaction-equilibrium parameter gaps are closed.

Use D2EHPA/TBP as the limitation baseline: it is produced-water relevant but the local model remains placeholder-heavy and weakly selective. Use TBP/FeCl3 and D2EHDTPA/BuPhen as backup chemistries if HBTA/TOPO parameter fitting stalls.

## Explicit ePC-SAFT Gap

The case study is scientifically strongest when it states the remaining gap plainly: the active objective is Li over Na after divalent pretreatment, and the Rezaee bridge is source-regressed and ePC-SAFT-supported rather than a fully accepted direct electrolyte-LLE phase split across the full Smackover surrogate domain.
